# Echoes of Jannah

A spiritual companion app that connects users with the Quran through emotions, reflections, and meaningful engagement.

## Features

- **Quran Journey** - Read, search, and listen to Quranic verses
- **Emotion Mirror** - Connect your emotions with divine guidance
- **Spiritual DNA** - Track your spiritual growth and achievements
- **Life Timeline** - Document life events with Quran connections
- **Community Hub** - Share reflections with fellow seekers
- **Daily Challenges** - Build consistent spiritual habits
- **Sacred Audio** - Listen to beautiful Quran recitations
- **Advanced Analytics** - Deep insights into your spiritual journey

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Quran Foundation API
- Recharts
